## Tooling & Productivity Guide
Stack enxuto para execução rápida em ambiente local.

## Required Tooling
- Node.js (Node 18+ recomendado)
- pnpm ou npm
- navegador moderno com suporte a SSE

## Recommended Automation
- `npm run dev` para rodar backend + UI.
- `npm run build` para build de verificação.

## IDE / Editor Setup
- Extensões úteis: ESLint/Prettier (opcional), React snippets, JSON formatter.

## Productivity Tips
- Deixar backend e UI em abas separadas para visualizar o loop em tempo real.
- Utilizar `localStorage` para preservar regras entre recargas durante testes.
